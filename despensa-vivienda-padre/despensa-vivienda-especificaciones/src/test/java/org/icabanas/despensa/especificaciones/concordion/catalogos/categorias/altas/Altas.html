Gestión de altas del catálogo de categorías

Como administrador del sistema, quiero gestionar las altas de las entidades Categoria.

Alta de una categoría.

Dado un usuario autenticado como adminstrador del sistema,
cuando da de alta la categoría Carnicería,
el sistema persiste la categoría que queda almacenada en el repositorio de almacenamiento.

Nombre Descripción
Carnicería

Alta de una categoría con descripción.

Dado un usuario autenticado como adminstrador del sistema,
cuando da de alta la categoría Lácteos con descripción Productos lácteos,
el sistema persiste la categoría que queda almacenada en el repositorio de almacenamiento.

Nombre Descripción
Lácteos Productos lácteos

Escenarios alternativos

  • Alta de categoría con nombre vacío
  • Alta de categoría con nombre nulo
  • Alta de categoría existente
