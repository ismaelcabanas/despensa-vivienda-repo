Gestión de bajas del catálogo de categorías

Como administrador del sistema, quiero gestionar las bajas de las entidades Categoria.

Supongamos que en el sistema existen las siguientes categorías:
Categoría
Lácteos
Frutas y Verduras

Baja de una categoría.

Cuando se da de baja la categoría Lácteos, ésta se da de baja del repositorio de almacenamiento y en el sistema quedarían las siguientes categorías:

Nombre
Frutas y Verduras
