Alta de categoría existente

El sistema debería validar que se da de alta una categoría existente.

Supongamos que en el sistema existe la siguiente categoría:
Categoría
Lácteos

Alta de una categoría con un nombre que ya existe.

Dado un usuario autenticado como adminstrador del sistema,
cuando da de alta una categoría con nombre Lácteos, entonces el sistema genera una excepción de validación y no almacena la entidad en el repositorio de datos.
