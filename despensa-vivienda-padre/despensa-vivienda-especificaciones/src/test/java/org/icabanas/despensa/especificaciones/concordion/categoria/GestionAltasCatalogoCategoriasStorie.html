Gestión de altas del catálogo de categorías

Como administrador del sistema, quiero gestionar las altas de las entidades Categoria.

Alta de una categoría.

Dado un usuario autenticado como adminstrador del sistema,
cuando da de alta la categoría Lacteos, entonces el sistema registra la categoría que queda almacenada en el repositorio de datos.

Alta de una categoría con nombre de categoría vacío.

Dado un usuario autenticado como adminstrador del sistema,
cuando da de alta una categoría con nombre vacío, entonces el sistema genera una excepción de validación y no almacena la entidad en el repositorio de datos.

Alta de una categoría con nombre nulo.

Dado un usuario autenticado como adminstrador del sistema,
cuando da de alta una categoría con nombre null, entonces el sistema genera una excepción de validación y no almacena la entidad en el repositorio de datos.
